## Supported Input Types
### **Commonly Used Types**
| Type    | Example Input          | Example Conversion                   |
|---------|------------------------|--------------------------------------|
| `int`   | `"42"`                 | `42`                                 |
| `float` | `"3.14"`               | `3.14`                               |
| `str`   | `"hello"`              | `"hello"`                            |
| `bool`  | `"True"` / `"False"`   | `True` / `False`                     |
| `list ` | `"1,2,3"`              | `[1, 2, 3]`                          |
| `tuple` | `"1,hello,3.5"`        | `(1, "hello", 3.5)`                  |
| `set`   | `"1,2,3"`              | `{1, 2, 3}`                          |
| `Any`   | `"anything"`           | `"anything"` (no conversion applied) |
