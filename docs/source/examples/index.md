# Examples

These examples are for everyone: users who want to see what IvoryOS can operate, integrators looking for platform patterns, and developers looking for real-world context.

Additional community integrations are shared in the [IvoryOS Hub gallery](https://ivoryos.ai/hub/gallery). The gallery includes platform, device, module, template, and community showcase posts.

## ADC

Antibody Drug Conjugation ([ADC](https://gitlab.com/heingroup/adc-automation)) platform.

## PurPOSE

Purification Platform Optimizing Solubility based Experimentation ([PurPOSE](https://gitlab.com/heingroup/purpose)).

## Flow Chem

The [Flow Chem](https://github.com/jiayu423/Autonomous-flow-optimizer) code was optimized for IvoryOS in [`flow_so_ivoryos`](https://github.com/ivoryzh/Autonomous-flow-optimizer/blob/main/single%20objective%20edbo/flow_so_ivoryos.py).

## LLE

Liquid-Liquid Extraction ([LLE](https://gitlab.com/heingroup/automated-lle)) is a work-in-progress platform for sampling biphasic reactions. The code repository may be unavailable at this moment.
